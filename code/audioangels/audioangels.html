Logline

A real-time “biblically accurate angel” audiovisualizer: low-poly forms, ritual geometry, and unstable analog artifacts synchronized to generative audio.

What it is

Built in ChucK + ChuGL, Audio Angels is a live audiovisual program that treats sound as a liturgy: drones, pulses, and harmonic tension drive a shifting celestial architecture—equal parts reverent and wrong.

Highlights

  • Real-time audio → visual mapping (intensity, spectral energy, rhythmic events)
  • Low-poly “angel” geometry + orbiting motifs (wheels / eyes / halos / rings)
  • Analog horror treatment: scanlines, bloom, jitter, vignette, “tape” noise
  • Modes / states (e.g., Heaven / Earth / Hell) with distinct palettes + motion rules

How to run (macOS)

  1. Download the project ZIP.
  2. Unzip the folder.
  3. Double-click Executable.command.
  4. Use the on-screen instructions (or your controls) to switch modes / trigger cues.
