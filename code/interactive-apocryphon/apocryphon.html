THE APOCRYPHON OF JOHN

Interactive audiovisual narrative (ChuGL) — Gnostic-inspired — 2025

Role: Creator · Tools: ChuGL, ChucK · Format: Interactive audiovisual narrative

HIGHLIGHTS

  • Scene-based interactive structure with mini-games gating progression
  • Procedural camera movement + player navigation
  • Real-time audio + visual synchronization
  • Symbolic “cosmology” environments (Pleroma / Sophia / Demiurge motifs)

HOW TO RUN (macOS)

  1. Download the project ZIP file.
  2. Unzip the folder.
  3. Double-click Executable.command.
  4. Use WASD / arrow keys to move, the keyboard to type text, and the cursor to click on objects.

LOGLINE

A (minimally) interactive audiovisual exposé on the Apocryphon of John .

WHAT IT IS

Built in ChuGL (ChucK + graphics), the project blends narrative animation with interaction. The player progresses through scenes inspired by the Apocryphon of John, completing small tasks to advance the story.

Script, music, and pixel art are all original. Voiced by Amy Wang and Komi Ergashev.

MOTIVATION

I wanted to treat a text like an explorable space: not an adaptation that “summarizes,” but a system that lets tone, symbolism, and constraint do narrative work.

The goal was to build something emotionally legible while staying formally strange—where meaning emerges through repeated gestures, small tasks, and audiovisual motifs.
