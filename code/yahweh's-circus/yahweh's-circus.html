Yahweh’s Circus

Interactive world (ChuGL) — surreal systems — 2025

Role: Creator · Tools: ChuGL, ChucK · Format: Interactive environment

Logline

A surreal, game-like “holy carnival” where mythic symbols become interactive mechanics—part ritual, part playground.

What it is

Yahweh’s Circus is an interactive audiovisual world built in ChuGL. The player navigates a sequence of spaces and encounters that behave like a system—objects orbit, breathe, trigger sound and text overlays, and rearrange the scene based on attention and movement.

The project explores how interface, spectacle, and narrative can merge into a playable environment—less “story you read,” more “story you inhabit.”

Highlights

  • Scene system + interactive triggers (objects, zones, timed events)
  • Procedural camera movement / motion motifs (orbits, breathing, drift)
  • Audio-driven mood shifts + reactive visuals
  • Stylized worldbuilding: carnival aesthetics, sacred geometry, surreal props

How to run (macOS)

  1. Download the project ZIP file.
  2. Unzip the folder.
  3. Double-click Executable.command.
  4. Use WASD / arrow keys to move, and follow on-screen prompts.
